# Abu Dhabi Public Transport Analysis Tool

## Overview
This tool analyzes public transport ridership and performance metrics in Abu Dhabi. It processes data to produce actionable insights for urban planners, researchers, and policymakers. The tool includes:

- Monthly ridership trends.
- Ridership distribution by route.
- Peak travel hours.

## Requirements

- Python 3.8+
- pandas
- matplotlib
- dash

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repo/public-transport-analysis.git
   
2. Navigate to the project directory:
   bash
   cd public-transport-analysis
   
3. Install dependencies:
   bash
   pip install -r requirements.txt
   

## Usage
1. Place your dataset in the project folder and update the file path in the script.
2. Run the script:
   bash
   python app.py
   
3. Open the dashboard in your web browser at `http://127.0.0.1:8050`.

## Features
- **Monthly Ridership Trends:** Bar graph showing ridership trends over months.
- **Ridership by Route:** Bar graph illustrating ridership distribution across routes.
- **Peak Travel Hours:** Line graph identifying peak usage hours.
